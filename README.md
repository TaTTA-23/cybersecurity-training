# Projeto: Monitorização de Segurança em Home Lab

Este repositório contém ferramentas de automação desenvolvidas para monitorizar a segurança de ativos em ambiente Linux/Windows.

## 🛠️ Tecnologias Utilizadas
- **Linguagens:** Python 3.x, Bash Scripting.
- **Ambiente:** Ubuntu Server / Kali Linux.

## 🚀 Projetos Incluídos

### 1. Detetor de Alterações em Ficheiros (Python)
- **O que faz:** Monitoriza em tempo real se ficheiros de configuração foram editados.
- **Valor para o Negócio:** Previne ataques de persistência e modificação de dados críticos.

### 2. Analisador de Força Bruta (Bash)
- **O que faz:** Varre os logs do sistema e identifica ataques de dicionário ao SSH.
- **Valor para o Negócio:** Reduz o ruído de logs e permite o bloqueio rápido de IPs maliciosos.

## 🛡️ Relação com Conformidade (Compliance)
Estes scripts auxiliam no cumprimento dos controlos da **ISO/IEC 27001** (Controlo de Logs e Monitorização) e estão alinhados com as diretrizes da **NIS2** para deteção proativa de incidentes.
